# 一、Elasticsearch简介

- 主要功能
  - 分布式搜索引擎
  - 大数据近实时分析引擎
- 产品特性
  - 高性能，和T+1说不
  - 容易使用/容易扩展

## （一）分布式架构

- 集群规模可以从单个扩展至数百个节点
- 高可用 & 水平扩展
  - 服务和数据两个维度
- 支持不同的节点类型
  - 支持Hot & Warm架构

## （二）多种方式集成接入

- 多种编程语言的类库
  - Java/.NET/Python/Ruby/PHP/Groovy/Perl
- RESTful API v.s Transport API
  - 9200 v.s 9300（建议使用RESTful API）
- JDBC & ODBC

## （三）主要功能

- 海量数据的分户式存储以及集群管理
  - 服务与数据的高可用，水平扩展
- 近实时搜索、性能卓越
  - 结构化/全文/地理位置/自动完成
- 海量数据的近实时分析
  - 聚合功能
